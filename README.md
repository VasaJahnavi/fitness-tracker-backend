## ✨ Features

- 🔐 **Secure Authentication** - JWT-based authentication with password hashing
- 👤 **Profile Management** - User profiles with fitness preferences and health metrics
- 💪 **Exercise Library** - Comprehensive exercise database with categories and muscle groups
- 📋 **Workout Plans** - Create, manage, and track workout routines
- 🎯 **Goal Tracking** - Set and monitor fitness goals with progress tracking
- 📊 **Activity Logging** - Log workouts with detailed exercise information
- 📈 **Analytics** - Track progress with detailed analytics and reports
- 🤖 **Recommendations** - Personalized workout recommendations based on user data
- 📱 **Dashboard** - User dashboard with key metrics and summary
- 📚 **API Documentation** - Complete Swagger/OpenAPI documentation

## 🛠️ Tech Stack

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB ODM

### Security
- **JWT** - JSON Web Tokens for authentication
- **Bcrypt** - Password hashing
- **Helmet** - Security headers
- **CORS** - Cross-Origin Resource Sharing

### Validation & Documentation
- **Joi** - Schema validation
- **Express Validator** - Request validation
- **Swagger** - API documentation

### Development
- **Nodemon** - Auto-reload during development
- **Morgan** - HTTP logging
- **Postman** - API testing
